# Installing on Mozilla Firefox

Before installing the userscript, you'll need a UserScript manager to manage it.

::: steps

1. Install Violentmonkey

   Install the Violentmonkey extension from the Chrome Web Store.

   <DownloadButton link="https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/">Get Violentmonkey</DownloadButton>

2. Install the userscript

   Once Violentmonkey is installed, install the OhMyMathpad userscript.

   <DownloadButton link="https://raw.githubusercontent.com/ld3z/omm/main/oh-my-mathpad.user.js" secondary>Download</DownloadButton>

:::

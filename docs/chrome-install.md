# Installing on Chrome

Before installing the userscript, you'll need a UserScript manager to manage it.

::: steps

1. Install ScriptCat

   Install the [ScriptCat extension](https://chromewebstore.google.com/detail/scriptcat/ndcooeababalnlpkfedmmbbbgkljhpjf) from the Chrome Web Store.

2. Install the userscript

   Once ScriptCat is installed, install the OhMyMathpad userscript.

:::
